# Prompt Library

Saved prompts for the 3-role workflow. Paste these at the start of a new chat
to set role + context. Update as you learn what works.

---

## PLANNER

**When to use:** Start of a task. Read-only chat. No code edits.

**How to use:** Open a new chat. Paste the prompt below. The agent reads inputs
and writes `.agent/plan.md`. You review and approve before moving to Implementer.
